Bumper Monkeys

Game Idea

This game is called Bumper Monkeys. As it says in the name, it includes both monkeys and bumper cars. This game allows for the user to experience the best of both worlds. The game will have singleplayer versus an AI as well as local multiplayer. The gameplay includes two monkeys in bumper cars and the objective is to knock the other player off the platform while you stay on. There will be a score calculated based on damage done, kills, and powerups. When you die 3 times, the game ends.

Screens

There will be a total of eight different screens. These screens include: Main Menu, Help, Input one name, Input two names, Singleplayer, Multiplayer, Win screen, and scoreboard. The menu screen greets the user and will link to each of the other screens. The help screen is for instructions on how to play the game. Input one name is for inputting one name for singleplayer. Input two names is for inputting two names for multiplayer. Singleplayer is the gameplay with an AI. Multiplayer is when two people want to verse each other on one computer. Next is the win screen, this displays who won. Finally, leaderboard is where session scores are kept.
